# Simple PHP MVC
Il s'agit d'un exemple de code pour un projet MVC en PHP de manière simple.

## Installation
1. Cloner le projet
2. Lancer votre serveur local
3. Aller sur le localhost dans votre navigateur

## Fonctionnalités
- Réécriture d'URL
- Gestion des erreurs
